# Work Item 105: PersistenceFacade.log_event() method

## Problem
`EventProcessor._route_event` checks `hasattr(self._persistence, "log_event")` and routes to it if present, but `PersistenceFacade` has no `log_event` method. The persistence routing path is dead code.

## Fix
Add to `PersistenceFacade`:

```python
from hamlet.event_processing.internal_event import InternalEvent

async def log_event(self, event: InternalEvent) -> None:
    """Log an event to the event log. Called by EventProcessor."""
    try:
        entry = EventLogEntry(
            timestamp=event.received_at,
            session_id=event.session_id or "",
            project_id=event.project_id or "",
            hook_type=event.hook_type.value,
            tool_name=event.tool_name,
            summary=f"{event.hook_type.value}: {event.tool_name or ''}",
        )
        await self.append_event_log(entry)
    except Exception:
        logger.error("Failed to log event to persistence", exc_info=True)
```
