# Implementation Notes: 082 - Event Processing Tests

## Scope

Test EventProcessor main loop and routing, EventRouter handler management, SequenceGenerator, and InternalEvent validation.

## Testing Patterns

### EventProcessor Tests

Mock the dependencies:

```python
@pytest.fixture
def processor():
    return EventProcessor()

@pytest.fixture
def mock_handlers():
    return {
        'world_state': AsyncMock(),
        'agent_inference': AsyncMock(),
        'persistence': AsyncMock(),
    }
```

Test validation:

```python
def test_validate_event():
    event = InternalEvent(
        id="test",
        session_id="",  # Invalid
        project_id="proj",
        hook_type=HookType.PreToolUse,
        tool_name="Read",
        tool_input={},
        received_at=datetime.now(UTC),
    )
    # Should raise validation error
```

Test routing with asyncio.gather:

```python
async def test_route_event():
    with patch.object(processor, '_handlers', mock_handlers):
        await processor.route_event(event)
        mock_handlers['world_state'].assert_called_once()
        mock_handlers['agent_inference'].assert_called_once()
```

Test consume loop:

```python
async def test_consume_loop():
    queue = asyncio.Queue()
    await queue.put(event)

    # Run one iteration
    task = asyncio.create_task(processor._consume())
    await asyncio.sleep(0.1)
    task.cancel()

    # Verify processed
    assert queue.empty()
```

### EventRouter Tests

Test subscribe/unsubscribe:

```python
def test_subscribe():
    router = EventRouter()
    callback = Mock()
    router.subscribe(callback)
    assert callback in router._subscribers

def test_unsubscribe():
    router.unsubscribe(callback)
    assert callback not in router._subscribers
```

### SequenceGenerator Tests

Test monotonic increments:

```python
async def test_sequence():
    gen = SequenceGenerator()
    seq1 = await gen.next()
    seq2 = await gen.next()
    assert seq2 == seq1 + 1
```

Test async safety with concurrent access.

### InternalEvent Tests

Test validation in __post_init__:

```python
def test_invalid_event():
    with pytest.raises(ValueError):
        InternalEvent(
            id="",  # Empty
            session_id="sess",
            project_id="proj",
            hook_type=HookType.PreToolUse,
        )
```

## GP-7 Error Handling

Verify errors in handlers don't crash the router:

```python
async def test_handler_error_logged():
    mock_handlers['world_state'].side_effect = Exception("boom")
    await processor.route_event(event)  # Should not raise
    # Verify error logged
```

## Key Test Scenarios

1. **Event validation** - Required fields, valid hook types
2. **Event routing** - All handlers called
3. **Handler errors** - Logged, not raised
4. **Consume loop** - Queue processing
5. **Router subscribe/unsubscribe** - Callback management
6. **Sequence generation** - Monotonic, thread-safe
7. **InternalEvent creation** - Validation, defaults
