# Implementation Notes: 088 - World State Tests

## Scope

Test PositionGrid (spatial tracking) and WorldStateManager CRUD operations, loading, and queries.

## Testing Patterns

### PositionGrid Tests

Test occupancy:

```python
def test_occupy():
    grid = PositionGrid()
    grid.occupy("agent1", Position(10, 10))
    assert grid.is_occupied(Position(10, 10))
    assert grid.get_entity_at(Position(10, 10)) == "agent1"
```

Test conflict handling:

```python
def test_occupy_conflict():
    grid.occupy("agent1", Position(10, 10))
    with pytest.raises(ValueError):
        grid.occupy("agent2", Position(10, 10))
```

### WorldStateManager Tests

Use real in-memory state or mocks:

```python
@pytest.fixture
def manager():
    persistence = Mock()
    return WorldStateManager(persistence)
```

Test CRUD:

```python
async def test_create_project():
    project = await manager.get_or_create_project("proj1", "Test Project")
    assert project.id == "proj1"
    assert project.name == "Test Project"
```

Test loading:

```python
async def test_load_from_persistence():
    data = WorldStateData(
        projects=[{"id": "p1", "name": "Project 1", "config_json": {}}],
        sessions=[],
        villages=[],
        agents=[],
        structures=[],
        metadata={}
    )
    await manager.load_from_persistence(data)
    assert "p1" in manager._state.projects
```

Test queries:

```python
async def test_agents_in_view():
    # Add agents at positions
    bounds = BoundingBox(0, 0, 100, 100)
    agents = await manager.get_agents_in_view(bounds)
    # Verify only agents in bounds returned
```

## Key Test Scenarios

1. **Grid occupy** - Stores entity at position
2. **Grid conflict** - Raises on duplicate
3. **Grid vacate** - Removes entity
4. **Grid is_occupied** - Boolean check
5. **Manager create project** - New project + village
6. **Manager create agent** - Spawn near parent
7. **Manager update agent** - Fields modified
8. **Manager load from persistence** - Restores all entities
9. **Manager agents in view** - Bounds filtering
10. **Manager structures in view** - Bounds filtering
