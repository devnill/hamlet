## src/hamlet/world_state/manager.py

Add method after `get_agents_in_view`:

```python
async def get_nearest_village_to(self, x: int, y: int) -> Village | None:
    """Return the village whose center is nearest to (x, y), or None if no villages."""
    import math
    async with self._lock:
        villages = list(self._state.villages.values())
    if not villages:
        return None
    return min(villages, key=lambda v: math.hypot(v.center.x - x, v.center.y - y))
```

## src/hamlet/tui/remote_world_state.py

Add method (client-side calculation from cached state):

```python
async def get_nearest_village_to(self, x: int, y: int) -> Village | None:
    """Return nearest village by Euclidean distance from cached villages."""
    import math
    if not self._villages:
        return None
    return min(self._villages, key=lambda v: math.hypot(v.center.x - x, v.center.y - y))
```

Note: `self._villages` is the cached list of villages from the most recent state poll.
