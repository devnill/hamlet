# Implementation Notes: 089 - MCP Server Tests

## Scope

Test validation (validate_event), handlers (event handlers), and MCPServer lifecycle.

## Testing Patterns

### Validation Tests

Test valid payloads:

```python
def test_validate_valid():
    payload = {
        "jsonrpc": "2.0",
        "method": "hamlet/event",
        "params": {
            "session_id": "sess1",
            "project_id": "proj1",
            "hook_type": "PreToolUse",
            "timestamp": "2024-01-01T00:00:00Z"
        }
    }
    result = validate_event(payload)
    assert result.valid
```

Test invalid payloads:

```python
def test_validate_missing_jsonrpc():
    payload = {"method": "hamlet/event", "params": {}}
    result = validate_event(payload)
    assert not result.valid
    assert "jsonrpc" in result.error
```

### Handler Tests

Test event handling:

```python
async def test_handle_event():
    queue = asyncio.Queue()
    payload = {...}  # Valid event

    await handle_hamlet_event(payload, queue)

    assert queue.qsize() == 1
    event = await queue.get()
    assert event["session_id"] == "sess1"
```

Test error handling (GP-7):

```python
async def test_handle_event_error():
    with patch('hamlet.mcp_server.handlers.logger') as mock_log:
        await handle_hamlet_event(invalid_payload, queue)
        mock_log.warning.assert_called()
```

### MCPServer Tests

Test lifecycle:

```python
async def test_server_lifecycle():
    server = MCPServer()
    await server.start()
    assert server.is_running()

    await server.stop()
    assert not server.is_running()
```

Test idempotent start/stop:

```python
async def test_start_idempotent():
    await server.start()
    await server.start()  # Second call should not fail
```

## Key Test Scenarios

1. **Validation valid** - Accepts correct payload
2. **Validation invalid jsonrpc** - Rejects missing
3. **Validation invalid method** - Rejects unknown
4. **Validation invalid hook_type** - Rejects unknown
5. **Handler enqueue** - Valid event queued
6. **Handler error** - Invalid logged, not raised
7. **Tool handler** - hamlet_status returns queue size
8. **Resource handler** - hamlet://world returns summary
9. **Server start** - Creates background task
10. **Server stop** - Cancels task gracefully
