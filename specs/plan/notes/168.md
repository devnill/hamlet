# WI-168 Notes: Register hamlet in claude-marketplace

## File to modify
`~/code/claude-marketplace/.claude-plugin/marketplace.json`

## What to add
Append to the `plugins` array:
```json
{
  "name": "hamlet",
  "source": {
    "source": "github",
    "repo": "devnill/hamlet"
  },
  "description": "Visualize Claude Code agent activity as a roguelike idle game.",
  "version": "0.1.0"
}
```

## Version note
Version `0.1.0` matches the hamlet repo's `.claude-plugin/plugin.json`. The repo is not yet published on GitHub — the entry is added now so it is ready when published. The `release.sh sync` command can update the version later if needed.

## Do not run release.sh add
The `add` subcommand fetches the remote plugin.json from GitHub. Since the repo is not yet public, this would fail. Edit marketplace.json directly.
