# Implementation Notes: 087 - TUI Integration Tests

## Scope

Test WorldView rendering and HamletApp integration using Textual's testing framework.

## Testing Patterns

### WorldView Tests

Mock world state:

```python
@pytest.fixture
def world_view():
    viewport = Mock()
    world_state = Mock()
    return WorldView(viewport, world_state)
```

Test render with agents:

```python
def test_render_agents():
    agent = Mock(
        position=Position(10, 10),
        state=AgentState.ACTIVE,
        color="cyan"
    )
    world_state.get_agents_in_view.return_value = [agent]

    text = world_view.render()
    assert "-" in text or "|" in text  # Spin symbol
```

### HamletApp Tests

Use TestApp for full integration:

```python
async def test_app():
    class TestHamletApp(HamletApp):
        def __init__(self):
            super().__init__(Mock(), Mock())

    async with TestHamletApp().run_test() as pilot:
        # Test actions
        await pilot.press("h")
        await pilot.pause()
```

Test actions:

```python
async def test_scroll_action():
    with patch.object(app._viewport, 'scroll') as mock_scroll:
        await pilot.press("h")
        mock_scroll.assert_called_with(-1, 0)
```

Test state updates:

```python
async def test_update_state():
    with patch.object(app, '_world_state') as mock_ws:
        mock_ws.get_agents_in_view.return_value = [Mock(), Mock()]
        await app._update_state()

        status = app.query_one(StatusBar)
        assert status.agent_count == 2
```

## Key Test Scenarios

1. **WorldView render agents** - Symbols at positions
2. **WorldView render structures** - Structure symbols
3. **WorldView empty cells** - Dots for floor
4. **WorldView animation** - Spin symbols for active
5. **App compose** - All widgets mounted
6. **App scroll actions** - hjkl bindings
7. **App toggle legend** - Visibility toggled
8. **App toggle follow** - Follow mode switched
9. **App update state** - Status bar updated
