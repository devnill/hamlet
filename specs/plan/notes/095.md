# Implementation Notes: 095 - Application Wiring and Orchestration

## Scope

Wire all components together in the main entry point.

## Component Initialization Order

```
1. Config (Settings.load())
2. PersistenceFacade (needs config.db_path)
3. WorldStateManager (needs persistence)
4. EventProcessor (needs world_state)
5. SimulationEngine (needs world_state)
6. MCPServer (needs event_processor)
7. HamletApp (needs all of above)
```

## Implementation

### src/hamlet/__main__.py (revised)

```python
import asyncio
import signal
from hamlet.config.settings import Settings
from hamlet.persistence.facade import PersistenceFacade
from hamlet.world_state.manager import WorldStateManager
from hamlet.event_processing.event_processor import EventProcessor
from hamlet.simulation.engine import SimulationEngine
from hamlet.mcp_server.server import MCPServer
from hamlet.viewport.manager import ViewportManager
from hamlet.tui.app import HamletApp

async def main_async():
    settings = Settings.load()

    # Initialize persistence
    persistence = PersistenceFacade(db_path=settings.db_path)
    await persistence.start()

    # Initialize world state
    world_state = WorldStateManager(persistence)
    await world_state.load_from_persistence(await persistence.load_state())

    # Initialize event processing
    event_processor = EventProcessor()

    # Initialize simulation
    simulation = SimulationEngine(world_state)
    await simulation.start()

    # Initialize MCP server
    mcp_server = MCPServer(event_queue=event_processor.get_queue())
    await mcp_server.start()

    # Wire event processor to world state
    event_processor.set_world_state(world_state)
    await event_processor.start()

    # Create viewport
    viewport = ViewportManager(world_state)

    # Run TUI
    app = HamletApp(world_state, viewport, event_processor)

    try:
        await app.run_async()
    finally:
        # Shutdown in reverse order
        await event_processor.stop()
        await mcp_server.stop()
        await simulation.stop()
        await persistence.stop()

def main():
    try:
        asyncio.run(main_async())
    except KeyboardInterrupt:
        print("\nHamlet stopped.")
    except Exception as e:
        print(f"Error: {e}")
        raise

if __name__ == "__main__":
    main()
```

- Component start failure → Log error, continue with degraded functionality (GP-7)
- Shutdown failure → Log warning, don't hang
- TUI crash → Clean up all components
