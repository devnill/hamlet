# Implementation Notes: 084 - Simulation Features Tests

## Scope

Test ExpansionManager (village expansion, road building) and AnimationManager (frame management, symbol/color generation).

## Testing Patterns

### ExpansionManager Tests

Mock WorldStateManager:

```python
@pytest.fixture
def expansion_mgr():
    world_state = Mock()
    return ExpansionManager(world_state)
```

Test expansion triggering:

```python
def test_expansion_triggered():
    village = Mock()
    village.agent_ids = ["a1", "a2", "a3"]  # Above threshold

    with patch.object(expansion_mgr, '_find_clear_site') as mock_find:
        mock_find.return_value = Position(10, 10)
        result = expansion_mgr._check_village_expansion(village)
        assert result == Position(10, 10)
```

Test road creation:

```python
def test_create_roads():
    village1 = Mock(center=Position(0, 0))
    village2 = Mock(center=10, 10)

    with patch.object(world_state, 'create_structure') as mock_create:
```

Test Bresenham line algorithm:

```python
def test_bresenham():
    points = expansion_mgr._bresenham(0, 0, 3, 3)
    assert (0, 0) in points
    assert (3, 3) in points
```

### AnimationManager Tests

Test state-based animation selection:

```python
def test_active_animation():
    agent = Mock()
    agent.state = AgentState.ACTIVE

    anim = manager.get_animation_state(agent)
    assert anim["type"] == "spin"
```

Test symbol cycling:

```python
def test_symbol_cycle():
    agent = Mock(state=AgentState.ACTIVE)

    symbols = []
    for i in range(8):
        state = manager.get_animation_state(agent)
        symbols.append(state["symbol"])
        manager.advance_frames([agent], 1)

    # Verify cycle repeats
    assert symbols[0] == symbols[4]
```

Test color generation:

```python
def test_zombie_color():
    agent = Mock(state=AgentState.ZOMBIE, color="cyan")

    state = manager.get_animation_state(agent)
    assert "green" in state["color"]
```

## Key Test Scenarios

1. **Expansion threshold** - Triggered when agents > threshold
2. **Clear site finding** - Finds unobstructed position
3. **Road creation** - Bresenham line between villages
4. **Animation types** - Spin, pulse, idle per state
5. **Symbol cycling** - SPIN_SYMBOLS rotation
6. **Color effects** - Zombie green tint
7. **Frame advancement** - Counters increment
8. **Idle cleanup** - Remove idle agents from frames
