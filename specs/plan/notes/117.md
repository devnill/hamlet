# WI-117: Move TOOL_TO_STRUCTURE to world_state/rules.py

## Context

WI-108 (Cycle 003) claimed to consolidate structure rules into `world_state/rules.py`. It moved `STRUCTURE_RULES` and `STRUCTURE_WORK_REQUIRED` successfully but left `TOOL_TO_STRUCTURE` as a local module-level dict in `inference/engine.py:28-35`. This violates GP-4 (Modularity): the canonical location for structure mappings is `world_state/rules.py`, not the inference engine.

## Current state

`inference/engine.py:28-35`:
```python
TOOL_TO_STRUCTURE: dict[str, str] = {
    "Read": "library",
    "Write": "workshop",
    "Edit": "workshop",
    ...
}
```

`world_state/rules.py`:
```python
__all__ = ["STRUCTURE_RULES"]

STRUCTURE_RULES = {
    StructureType.LIBRARY: {...},
    ...
}
```

## Fix

**Step 1**: In `world_state/rules.py`, add the `TOOL_TO_STRUCTURE` dict. It must reference `StructureType` enum values (from `world_state.types`), not raw strings:

```python
from .types import StructureType

TOOL_TO_STRUCTURE: dict[str, StructureType] = {
    "Read": StructureType.LIBRARY,
    "Glob": StructureType.LIBRARY,
    "Grep": StructureType.LIBRARY,
    "Write": StructureType.WORKSHOP,
    "Edit": StructureType.WORKSHOP,
    "Bash": StructureType.FORGE,
    "Task": StructureType.FORGE,
    # ... include all entries from the current engine.py dict
}
```

Add `"TOOL_TO_STRUCTURE"` to `__all__`.

**Step 2**: In `inference/engine.py`, remove the local `TOOL_TO_STRUCTURE` definition and add an import:

```python
from hamlet.world_state.rules import TOOL_TO_STRUCTURE
```

**Step 3**: Verify all usages of `TOOL_TO_STRUCTURE` in `engine.py` still work after the type change (if values were strings before, they are now `StructureType` enum values — `add_work_units` in `manager.py` accepts `StructureType`, so this should be correct or may require updating the lookup).

## Key locations

- `engine.py:28-35` — local TOOL_TO_STRUCTURE definition (remove)
- `engine.py` — import section (add import from world_state.rules)
- `world_state/rules.py` — add TOOL_TO_STRUCTURE dict and __all__ entry
- `world_state/types.py` — StructureType enum (reference for valid values)
