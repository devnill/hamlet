# Work Item 102: Wire AgentInferenceEngine in __main__.py

## Problem
`AgentInferenceEngine` is fully implemented but never instantiated in `__main__.py`. Both `EventProcessor` and `SimulationEngine` accept it as a parameter but receive `None`, making the inference subsystem a no-op.

## Fix
After `world_state` is initialized, before creating `event_processor`:
```python
from hamlet.inference.engine import AgentInferenceEngine

agent_inference = AgentInferenceEngine(world_state)
```

Pass it to both:
```python
event_processor = EventProcessor(
    event_queue=event_queue,
    world_state=world_state,
    agent_inference=agent_inference,
    persistence=persistence,
)

simulation = SimulationEngine(
    world_state=world_state,
    agent_inference=agent_inference,
    ...
)
```

## Notes
Check whether `AgentInferenceEngine` has a `start()` / `stop()` lifecycle method that needs to be called in the startup/shutdown sequence. If so, add it in the correct order (after `world_state`, before `event_processor`).
