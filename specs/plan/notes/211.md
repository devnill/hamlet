## AgentInferenceEngine changes

The engine is in `src/hamlet/inference/engine.py`. It already has
`ZOMBIE_THRESHOLD_SECONDS = 300`. Add `DESPAWN_THRESHOLD_SECONDS` read from settings,
or pass the setting into the constructor.

### _handle_stop() — session end = immediate despawn

Current behavior: marks session agents IDLE when stop_reason is authoritative.
New behavior: despawns session agents immediately.

```python
# Replace the IDLE block (around line 380-386):
if session:
    for agent_id in session.agent_ids:
        try:
            await self._world_state.despawn_agent(agent_id)
            # Clean up inference engine state for this agent
            self._state.last_seen.pop(agent_id, None)
            self._state.zombie_since.pop(agent_id, None)
        except Exception:
            logger.exception("_handle_stop: failed to despawn agent %s", agent_id)
```

### _update_zombie_states() — record zombie_since + TTL despawn

After marking an agent ZOMBIE, record the transition time:
```python
if self._check_zombie(agent_id):
    await self._world_state.update_agent(agent_id, state=AgentState.ZOMBIE)
    if agent_id not in self._state.zombie_since:
        self._state.zombie_since[agent_id] = datetime.now(UTC)
```

Then add a despawn pass after the zombie-marking loop:
```python
despawn_cutoff = datetime.now(UTC) - timedelta(seconds=self.DESPAWN_THRESHOLD_SECONDS)
for agent_id in list(self._state.zombie_since.keys()):
    if self._state.zombie_since[agent_id] <= despawn_cutoff:
        try:
            await self._world_state.despawn_agent(agent_id)
            self._state.last_seen.pop(agent_id, None)
            self._state.zombie_since.pop(agent_id, None)
        except Exception:
            logger.exception("_update_zombie_states: failed to despawn agent %s", agent_id)
```

### Startup seeding of zombie_since

When the daemon starts, WI-208 loads all agents as ZOMBIE. The inference engine's
`zombie_since` is empty, so TTL despawn would never fire for them.

After `AgentInferenceEngine` starts (or in a `startup()` method), seed `zombie_since`
from world state:

```python
async def startup(self) -> None:
    """Seed zombie_since from agents already in ZOMBIE state at startup."""
    agents = await self._world_state.get_all_agents()
    for agent in agents:
        if agent.state == AgentState.ZOMBIE:
            # Use last_seen from DB as the zombie start time
            self._state.zombie_since[agent.id] = agent.last_seen
```

Call this from `SimulationEngine.start()` or `app_factory.py` after components are built.

### Resurrection

No special handling needed. If a despawned agent receives a new PreToolUse event,
`get_or_create_agent()` on WorldStateManager creates a fresh agent entity. The inference
engine treats it as a new agent.
