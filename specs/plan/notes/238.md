async def _load_world_metadata(self, cursor: sqlite3.Cursor) -> dict[str, str]:
    cursor.execute("SELECT key, value FROM world_metadata")
    return {row["key"]: row["value"] for row in cursor.fetchall()}

### src/hamlet/world_state/manager.py

Already done in WI-235 — terrain seed is read/written in `load_from_persistence`:

```python
# Reading
terrain_seed = self._state.world_metadata.get("terrain_seed")

# Writing (on first creation)
if terrain_seed is None:
    terrain_seed = str(random.randint(0, 2**31 - 1))
    self._state.world_metadata["terrain_seed"] = terrain_seed
    await self._persistence.queue_write("world_metadata", "terrain_seed", {"key": "terrain_seed", "value": terrain_seed})
```

## Acceptance Criteria

- Terrain seed is stored in `world_metadata` table with key `"terrain_seed"`
- On fresh start (no seed in DB), a random seed is generated and persisted
- On subsequent starts, the existing seed is loaded from DB
- Same world seed produces same terrain on restart
- A test verifies terrain seed persists across `load_state` / `queue_write` cycle
- A test verifies world produces same terrain after restart
