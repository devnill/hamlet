# WI-152: Fix deterministic agent color assignment

## Context
Agents are created with hardcoded `color="white"` in WorldStateManager instead of deriving color from `inferred_type`. TYPE_COLORS in inference/types.py (or world_state after WI-146) maps AgentType to color strings.

Violates GP-6 (Deterministic Agent Identity).

## Existing Code
- `src/hamlet/world_state/manager.py` — create_agent method (look for color="white")
- `src/hamlet/inference/types.py` — TYPE_COLORS dict (or world_state/types.py after WI-146)

## Fix
In manager.py create_agent, derive color from inferred_type:
```python
from hamlet.inference.types import TYPE_COLORS

color = TYPE_COLORS.get(inferred_type, "white")
```

## Notes
- Read manager.py create_agent method before changing
- Depends on WI-146 (AgentType consolidation) — TYPE_COLORS must reference consolidated AgentType
- inferred_type may default to AgentType.GENERAL for new agents
