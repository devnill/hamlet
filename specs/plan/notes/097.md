## Test Fixtures

```python
@pytest.fixture
async def e2e_components(tmp_path):
    """Create full component stack with temp database."""
    db_path = tmp_path / "test.db"
    persistence = PersistenceFacade(db_path=db_path)
    await persistence.start()

    world_state = WorldStateManager(persistence)
    event_processor = EventProcessor()
    simulation = SimulationEngine(world_state)
    mcp_server = MCPServer()

    yield {
        "persistence": persistence,
        "world_state": world_state,
        "event_processor": event_processor,
        "simulation": simulation,
        "mcp_server": mcp_server,
    }

    # Cleanup
    await mcp_server.stop()
    await simulation.stop()
    await event_processor.stop()
    await persistence.stop()
```

## Key Testing Patterns

1. **Use temp directories** - Don't pollute user's ~/.hamlet
2. **Use real SQLite** - Not mocks, verify actual persistence
3. **Use short timeouts** - E2E tests should be fast
4. **Test degradation** - Verify system continues when components fail (GP-7)

## Example Test

```python
async def test_multiple_sessions_create_multiple_agents(e2e_components):
    components = e2e_components

    # Simulate events from session 1
    await send_mock_hook_event(session_id="sess1", tool_name="Read")

    # Simulate events from session 2
    await send_mock_hook_event(session_id="sess2", tool_name="Write")

    # Process events
    await asyncio.sleep(0.1)  # Let pipeline process

    # Verify two agents exist
    agents = await components["world_state"].get_all_agents()
    assert len(agents) == 2
```
