# WI-150: Wire simulation components in __main__.py

## Context
`src/hamlet/__main__.py` creates SimulationEngine with only world_state; all other dependencies (agent_updater, structure_updater, expansion_manager, animation_manager, agent_inference) are None. The simulation tick loop runs but never updates agents or structures.

## Existing Code
- `src/hamlet/__main__.py` — creates SimulationEngine
- `src/hamlet/cli/commands/daemon.py` — creates SimulationEngine for daemon mode
- `src/hamlet/simulation/engine.py` — SimulationEngine class and its accepted params
- `src/hamlet/simulation/agent_updater.py` — AgentUpdater
- `src/hamlet/simulation/structure_updater.py` — StructureUpdater
- `src/hamlet/simulation/expansion.py` — ExpansionManager
- `src/hamlet/simulation/animation.py` — AnimationManager

## Changes
In both __main__.py and daemon.py, instantiate and pass all updaters:
```python
from hamlet.simulation.agent_updater import AgentUpdater
from hamlet.simulation.structure_updater import StructureUpdater
from hamlet.simulation.expansion import ExpansionManager
from hamlet.simulation.animation import AnimationManager

animation_manager = AnimationManager()
simulation = SimulationEngine(
    world_state=world_state,
    agent_updater=AgentUpdater(),
    structure_updater=StructureUpdater(),
    expansion_manager=ExpansionManager(),
    animation_manager=animation_manager,
)
```
