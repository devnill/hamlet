# Implementation Notes: 090 - Hook Scripts - PreToolUse and PostToolUse

## Scope

Create minimal client-side hook scripts that extract telemetry and send to the MCP server.

## Hook Script Requirements

### pre_tool_use.py

```python
#!/usr/bin/env python3
"""Send PreToolUse event to Hamlet MCP server."""
import json
import os
import urllib.request
import sys

SERVER_URL = "http://localhost:8080/hamlet/event"

def main():
    try:
        payload = {
            "jsonrpc": "2.0",
            "method": "hamlet/event",
            "params": {
                "hook_type": "PreToolUse",
                "session_id": os.environ.get("CLAUDE_SESSION_ID", "unknown"),
                "project_id": os.environ.get("CLAUDE_PROJECT_ID", "unknown"),
                "agent_name": os.environ.get("CLAUDE_AGENT_NAME", "unknown"),
                "tool_name": os.environ.get("CLAUDE_TOOL_NAME", ""),
                "timestamp": datetime.utcnow().isoformat()
            }
        }

        req = urllib.request.Request(
            SERVER_URL,
            data=json.dumps(payload).encode(),
            headers={"Content-Type": "application/json"},
            method="POST"
        )
        urllib.request.urlopen(req, timeout=1)
    except Exception:
        pass  # GP-7: fail silently
    finally:
        sys.exit(0)

if __name__ == "__main__":
    main()
```

### post_tool_use.py

Same structure but extracts `token_count` if available in environment.

## Key Design Decisions

1. **No external dependencies** - Use stdlib only for portability
2. **Hardcoded localhost:8080** - Server runs locally, no config needed
3. **1-second timeout** - Never block Claude Code
4. **Always exit 0** - Never fail the Claude Code operation
5. **Minimal payload** - Just what's needed for inference
