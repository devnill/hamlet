# Implementation Notes: 081 - Persistence Layer Data Tests

## Scope

Test WriteExecutor batch operations, StateLoader entity loading, WriteQueue batching, and EventLogManager append/prune.

## Testing Patterns

### WriteExecutor Tests

Test batch execution with real SQLite in memory:

```python
@pytest.fixture
async def executor():
    db = DatabaseConnection(":memory:")
    await db.__aenter__()
    await run_migrations(db)
    yield WriteExecutor(db)
    await db.__aexit__(None, None, None)
```

Test INSERT OR REPLACE for each entity type:
- Project with config_json
- Session with agent_ids_json
- Agent with all fields
- Structure with stage/material
- Village with bounds

Test DELETE operations via `_TABLE_MAP`.

Test rollback on error using side_effect=Exception.

### StateLoader Tests

Test loading from database with JSON fields parsed:

```python
# Insert test data
await db.execute("INSERT INTO projects ...", (...))

# Load and verify
loader = StateLoader(db)
state = await loader.load_state()
assert len(state.projects) == 1
assert state.projects[0]["config_json"] == {"key": "value"}
```

Test graceful degradation on missing table:

```python
# Drop table, then load
await db.execute("DROP TABLE projects")
state = await loader.load_state()
assert state.projects == []  # Empty list, not exception
```

### WriteQueue Tests

Test async queue operations:

```python
queue = WriteQueue(max_size=2)
await queue.put(op1)
await queue.put(op2)
# Third put should drop silently
await queue.put(op3)  # No exception
```

Test batch retrieval:

```python
batch = await queue.get_batch(max_items=2)
assert len(batch) == 2
```

Test task_done/join protocol:

```python
batch = await queue.get_batch()
for _ in batch:
    queue.task_done()
await queue.join()  # Should complete
```

### EventLogManager Tests

Test append with pruning:

```python
manager = EventLogManager(db, max_entries=5)
for i in range(10):
    await manager.append(entry)

# Verify only 5 entries remain
await db.execute("SELECT COUNT(*) FROM event_log")
row = await db.fetchone()
assert row[0] == 5
```

Test transaction rollback on error.

## Key Test Scenarios

1. **WriteExecutor batch insert** - All entity types
