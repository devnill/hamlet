# WI-236: Terrain symbols and colors for TUI

## Summary
Add terrain display symbols and colors to `tui/symbols.py` for use in WorldView rendering.

## Implementation

### src/hamlet/tui/symbols.py

Add terrain symbols and colors:

```python
# Terrain symbols (background layer)
TERRAIN_SYMBOLS = {
    "water": "~",
    "mountain": "^",
    "forest": "♣",
    "meadow": '"',
    "plain": ".",
}

TERRAIN_COLORS = {
    "water": "blue",
    "mountain": "grey85",
    "forest": "green",
    "meadow": "chartreuse",
    "plain": "white",
}


def get_terrain_symbol(terrain_type: str) -> str:
    """Return ASCII symbol for terrain type."""
    return TERRAIN_SYMBOLS.get(terrain_type, ".")


def get_terrain_color(terrain_type: str) -> str:
    """Return Rich color name for terrain type."""
    return TERRAIN_COLORS.get(terrain_type, "white")
```

## Acceptance Criteria

- `TERRAIN_SYMBOLS` dict has exactly 5 entries matching `TerrainType` values
- `TERRAIN_COLORS` dict has exactly 5 entries with valid Rich color names
- `get_terrain_symbol("water")` returns `"~"`
- `get_terrain_symbol("mountain")` returns `"^"`
- `get_terrain_color("water")` returns `"blue"`
- `get_terrain_color("forest")` returns `"green"`
- Unknown terrain types return `"."` symbol and `"white"` color
