```python
def load_settings() -> dict:
    if CLAUDE_SETTINGS_PATH.exists():
        return json.loads(CLAUDE_SETTINGS_PATH.read_text())
    return {}

def backup_settings(settings: dict):
    backup_path = CLAUDE_SETTINGS_PATH.with_suffix(".json.backup")
    backup_path.write_text(json.dumps(settings, indent=2))
```

## Error Handling (GP-11)

- Claude Code not installed → "Claude Code not found. Install from..."
- settings.json invalid → "Corrupted settings. Restore from backup at..."
- hooks directory not found → "Hamlet not properly installed. Reinstall with pip..."

## Uninstall Command

```bash
$ hamlet uninstall
```

Removes hooks from settings.json, restores from backup if exists.
