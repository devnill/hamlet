```bash

```

`BASH_SOURCE` is bash-only. mcp.json invokes via `"command": "sh"`. Fix options:
1. Change shebang to `#!/usr/bin/env bash` AND update mcp.json to use `"command": "bash"` instead of `"sh"`
2. Use `$0` instead: `SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"` — works in POSIX sh

Option 2 is simpler (no mcp.json change needed). Use `$0`.
