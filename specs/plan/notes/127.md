# WI-127: Create viewer mode and refactor entry point

## Context
The viewer starts only the TUI, polling the daemon's REST API for world state instead of holding a live in-memory WorldStateManager reference. It checks daemon health on startup and exits if the daemon isn't running.

## Existing Code to Reference
- `src/hamlet/tui/app.py` — `HamletApp`. Currently receives `world_state: WorldStateManager` and `viewport: ViewportManager`. The viewer passes a `RemoteWorldState` instead of a real WorldStateManager — both implement the same read interface.
- `src/hamlet/tui/world_view.py` — calls `world_state.get_agents_in_view(bounds)` and `world_state.get_structures_in_view(bounds)`
- `src/hamlet/tui/status_bar.py` — calls `world_state.get_projects()`, `world_state.get_all_agents()`, etc.
- `src/hamlet/tui/event_log.py` — calls `world_state.get_event_log()`
- `src/hamlet/world_state/types.py` — dataclass types that RemoteWorldState must reconstruct from JSON
- `src/hamlet/__main__.py` — current `_run_app()`, to be refactored

## New Files

### `src/hamlet/tui/remote_state.py` — RemoteStateProvider
```python
class RemoteStateProvider:
    def __init__(self, base_url: str = "http://localhost:8080"):
        self._base_url = base_url
        self._session: aiohttp.ClientSession | None = None

    async def start(self):
        self._session = aiohttp.ClientSession()

    async def stop(self):
        if self._session:
            await self._session.close()

    async def check_health(self) -> bool:
        try:
            async with self._session.get(f"{self._base_url}/hamlet/health", timeout=aiohttp.ClientTimeout(total=2)) as r:
                return r.status == 200
        except Exception:
            return False

    async def fetch_state(self) -> dict:
        async with self._session.get(f"{self._base_url}/hamlet/state") as r:
            return await r.json()

    async def fetch_events(self) -> list[dict]:
        async with self._session.get(f"{self._base_url}/hamlet/events") as r:
            data = await r.json()
            return data.get("events", [])
```

### `src/hamlet/tui/remote_world_state.py` — RemoteWorldState
```python
class RemoteWorldState:
    """WorldStateManager read interface backed by HTTP polling of the daemon."""

    def __init__(self, provider: RemoteStateProvider):
        self._provider = provider
        self._cached_state: dict = {}
        self._cached_events: list = []

    async def refresh(self):
        """Fetch fresh state from daemon. Called by background task in viewer."""
        try:
            self._cached_state = await self._provider.fetch_state()
            self._cached_events = await self._provider.fetch_events()
        except Exception:
            pass  # GP-7: keep serving stale cache on failure

    async def get_all_agents(self) -> list[Agent]:
        return [_deserialize_agent(a) for a in self._cached_state.get("agents", [])]

    async def get_all_structures(self) -> list[Structure]:
        return [_deserialize_structure(s) for s in self._cached_state.get("structures", [])]

    async def get_all_villages(self) -> list[Village]:
        return [_deserialize_village(v) for v in self._cached_state.get("villages", [])]

    async def get_projects(self) -> list[Project]:
        return [_deserialize_project(p) for p in self._cached_state.get("projects", [])]

    async def get_event_log(self, limit: int = 100) -> list:
        return self._cached_events[:limit]

    async def get_agents_in_view(self, bounds) -> list[Agent]:
        agents = await self.get_all_agents()
        return [a for a in agents if bounds.min_x <= a.position.x <= bounds.max_x
                and bounds.min_y <= a.position.y <= bounds.max_y]

    async def get_structures_in_view(self, bounds) -> list[Structure]:
        structures = await self.get_all_structures()
        return [s for s in structures if bounds.min_x <= s.position.x <= bounds.max_x
                and bounds.min_y <= s.position.y <= bounds.max_y]

    def get_animation_frame(self, agent_id: str) -> int:
        return self._cached_state.get("animation_frames", {}).get(agent_id, 0)
```

## __main__.py Refactor

Extract viewer mode into `_run_viewer(base_url: str)`:
```python
async def _run_viewer(base_url: str = "http://localhost:8080") -> None:
    provider = RemoteStateProvider(base_url)
    await provider.start()
    if not await provider.check_health():
        print("Hamlet daemon is not running. Start it with: hamlet daemon")
        await provider.stop()
        return

    remote_state = RemoteWorldState(provider)
    viewport = ViewportManager(world_state=remote_state)
    app = HamletApp(world_state=remote_state, viewport=viewport, remote_provider=provider)
    await app.run_async()
    await provider.stop()
```

The `HamletApp` refresh loop (`set_interval`) calls `remote_state.refresh()` to update cached state before each render pass. Add `remote_provider` param to `HamletApp.__init__` (optional, so existing tests pass).
