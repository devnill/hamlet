# WI-222 Implementation Notes

## Context

In `src/hamlet/inference/engine.py`, `_handle_stop` currently calls `despawn_agent` for all
stop reasons in {"tool", "stop", "end_turn"}.

Change: for stop_reason=="tool" (mid-tool interruption), transition agents to ZOMBIE instead
of despawning — they may resume. For clean stops, despawn immediately.

## Changes to `src/hamlet/inference/engine.py`

In `_handle_stop`, split the behavior on stop_reason:

```python
if event.stop_reason == "tool":
    # Flush pending tools (existing code)
    ...
    # Transition to ZOMBIE instead of despawn
    if session:
        for agent_id in list(session.agent_ids):
            try:
                await self._world_state.update_agent(agent_id, state=AgentState.ZOMBIE)
                self._state.zombie_since[agent_id] = datetime.now(UTC)
            except Exception:
                logger.exception("_handle_stop: failed to zombie agent %s", agent_id)
elif event.stop_reason in ("stop", "end_turn"):
    # Immediate despawn (existing behavior)
    if session:
        for agent_id in list(session.agent_ids):
            try:
                await self._world_state.despawn_agent(agent_id)
                self._state.last_seen.pop(agent_id, None)
                self._state.zombie_since.pop(agent_id, None)
            except Exception:
                logger.exception("_handle_stop: failed to despawn agent %s", agent_id)
# stop_reason==None: update last_seen only (existing behavior, no change)
```
