# Implementation Notes: 091 - Hook Scripts - Notification and Stop

## Scope

Complete the hook script suite with Notification and Stop handlers.

## Implementation

### notification.py

Similar structure to pre_tool_use.py but:
- `hook_type`: "Notification"
- Extracts `notification_type` and `message` from env
- Used for non-tool events (errors, completions, etc.)

### stop.py

- `hook_type`: "Stop"
- Signals session end
- Allows server to mark agents idle/zombie faster
