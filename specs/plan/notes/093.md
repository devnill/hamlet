# Implementation Notes: 093 - Application Entry Point

## Scope

Create the entry point to run Hamlet as a command-line tool.

## Implementation

### src/hamlet/__main__.py

```python
"""Entry point for hamlet package."""
import sys
import asyncio
from pathlib import Path

# Add src to path if running from source
def main():
    from hamlet.tui.app import HamletApp
    from hamlet.world_state.manager import WorldStateManager
    from hamlet.persistence.facade import PersistenceFacade
    from hamlet.event_processing.event_processor import EventProcessor
    from hamlet.simulation.engine import SimulationEngine
    from hamlet.mcp_server.server import MCPServer

    # Initialize components
    persistence = PersistenceFacade()
    world_state = WorldStateManager(persistence)
    event_processor = EventProcessor()
    simulation = SimulationEngine(world_state)
    mcp_server = MCPServer()

    # Wire dependencies
    # ...

    # Run TUI
    app = HamletApp(world_state, ...)
    app.run()

if __name__ == "__main__":
    main()
```

### pyproject.toml Update

```toml
[project.scripts]
hamlet = "hamlet.__main__:main"
```

## Design Decisions

1. **No argparse yet** - Keep simple, add CLI args in future work item if needed
2. **Single command** - Just `hamlet` runs the app
3. **Async entry** - May need `asyncio.run()` wrapper
4. **Error handling** - Catch and print friendly errors
