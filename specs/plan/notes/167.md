# WI-167 Notes: Remove inline hooks from ~/.claude/settings.json

## File to modify
`~/.claude/settings.json`

## Current state
settings.json has:
- `"hooks"` — four inline hook entries using `jq + curl` one-liners, one per hook type
- `"extraKnownMarketplaces"` — the devnill/claude-marketplace registration

## What to change
Remove the entire `"hooks"` key and its value from the JSON object. Keep `"extraKnownMarketplaces"` and any other top-level keys exactly as-is.

## Result
The plugin's hooks.json (WI-166) becomes the sole source of hook configuration once the hamlet plugin is installed. No inline hooks remain that could duplicate events or conflict.

## Note on notify.sh
`~/.claude/hooks/notify.sh` exists but is separate (likely from beepboop plugin). Do not touch it.
