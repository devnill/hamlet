## tests/test_event_processor.py

### Parametrized HookType test

```python
@pytest.mark.parametrize("hook_type_str", [ht.value for ht in HookType])
async def test_process_event_all_hook_types(hook_type_str):
    processor = EventProcessor(...)
    raw = {
        "session_id": "sess-1",
        "project_id": "proj-1",
        "hook_type": hook_type_str,
    }
    event = await processor.process_event(raw)
    assert event.hook_type == HookType(hook_type_str)
    assert event.session_id == "sess-1"
```
