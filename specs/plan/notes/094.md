# Implementation Notes: 094 - Configuration Module

## Scope

Create configuration system for user settings.

## Implementation

### src/hamlet/config/paths.py

```python
from pathlib import Path

HAMLET_DIR = Path.home() / ".hamlet"
CONFIG_PATH = HAMLET_DIR / "config.json"
DB_PATH = HAMLET_DIR / "world.db"

def ensure_hamlet_dir():
    HAMLET_DIR.mkdir(parents=True, exist_ok=True)
```

### src/hamlet/config/settings.py

```python
import json
from dataclasses import dataclass, asdict
from pathlib import Path
from typing import Optional

from .paths import CONFIG_PATH, ensure_hamlet_dir

@dataclass
class Settings:
    db_path: str = "~/.hamlet/world.db"
    mcp_port: int = 8080
    tick_rate: float = 30.0
    theme: str = "default"
    event_log_max_entries: int = 1000

    @classmethod
    def load(cls) -> "Settings":
        ensure_hamlet_dir()
        if CONFIG_PATH.exists():
            data = json.loads(CONFIG_PATH.read_text())
            return cls(**data)
        # Create defaults
        settings = cls()
        settings.save()
        return settings

    def save(self) -> None:
        ensure_hamlet_dir()
        CONFIG_PATH.write_text(json.dumps(asdict(self), indent=2))
```

## Design Principles

1. **Sensible defaults** - Works out of the box (GP-11)
2. **Auto-create config** - No manual setup needed
3. **JSON format** - Human editable
4. **Expand user paths** - `~` works in paths
