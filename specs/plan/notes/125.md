# WI-125: Add REST state endpoints to MCP server

## Context
The daemon/viewer split requires the daemon to expose world state over HTTP so the viewer can poll it. The existing `MCPServer` already has an aiohttp server started in `start()`. This work item adds two new routes to that server.

## Existing Code
- `src/hamlet/mcp_server/server.py` — `MCPServer` class. `__init__` takes `world_state: Any`. `start()` creates `aiohttp.web.Application`, adds routes (`/hamlet/event`, `/hamlet/health`), creates `TCPSite`. Add two new routes here.
- `src/hamlet/world_state/manager.py` — `WorldStateManager`. Methods: `get_all_agents()`, `get_all_structures()`, `get_all_villages()`, `get_projects()`, `get_event_log()`.
- `src/hamlet/world_state/types.py` — `Agent`, `Structure`, `Village`, `Project`, `Position`, `AgentType`, `AgentState`, `StructureType`, `ActivityType` dataclasses/enums.
- `src/hamlet/simulation/animation.py` — `AnimationManager`. Has `get_animation_frame(agent_id)` or similar. Check actual API.

## Implementation
1. Create `src/hamlet/mcp_server/serializers.py` with two functions:
   - `serialize_state(world_state, animation_manager=None) -> dict` — serializes all agents, structures, villages, projects, animation_frames
   - `serialize_events(world_state) -> dict` — serializes event log

2. In `MCPServer.__init__`, add optional `animation_manager: Any = None` parameter.

3. In `MCPServer.start()`, add routes:
   ```python
   app.router.add_get("/hamlet/state", self._handle_state)
   app.router.add_get("/hamlet/events", self._handle_events)
   ```

4. Add handlers:
   ```python
   async def _handle_state(self, request):
       try:
           data = serialize_state(self._world_state, self._animation_manager)
           return web.json_response(data)
       except Exception as e:
           return web.json_response({"error": str(e)}, status=500)
   ```

## Serialization Notes
- `datetime` → `.isoformat()`
- `Enum` → `.value`
- `Position(x, y)` → `{"x": pos.x, "y": pos.y}`
- `AgentState` enum → `.value` string
- `None` → `null` (JSON default)
- Animation frames: if `animation_manager` is None, return `{}`; otherwise call its method to get current frames for all visible agents.
