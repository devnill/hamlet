# WI-149: Wire AgentInferenceEngine to EventProcessor

## Context
`src/hamlet/__main__.py` initializes EventProcessor with `agent_inference=None`, breaking the event flow. AgentInferenceEngine is created elsewhere but never connected to the event pipeline. Same issue in daemon.py.

## Existing Code
- `src/hamlet/__main__.py` — creates EventProcessor
- `src/hamlet/cli/commands/daemon.py` — creates EventProcessor for daemon mode
- `src/hamlet/inference/engine.py` — AgentInferenceEngine class

## Changes

### Both __main__.py and daemon.py
Ensure AgentInferenceEngine is instantiated before EventProcessor:
```python
from hamlet.inference.engine import AgentInferenceEngine

agent_inference = AgentInferenceEngine(world_state)
event_processor = EventProcessor(
    event_queue=event_queue,
    world_state=world_state,
    agent_inference=agent_inference,
    persistence=persistence,
)
```

- Read both files before modifying
- Depends on WI-146 (AgentType consolidation) to avoid conversion errors
- AgentInferenceEngine constructor signature: check engine.py before coding
