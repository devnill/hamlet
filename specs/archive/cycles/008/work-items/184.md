# WI-184 Notes: Adaptive viewport

## Context
ViewportManager.resize(width, height) exists at viewport/manager.py:82 but is never called. ViewportState defaults to Size(80, 24). WorldView needs to call resize on mount and on Textual resize events.

## Implementation
In world_view.py:
1. Add: from textual.events import Resize
2. In on_mount(), after self.set_interval(1/4, self._update_animation_frame):
   self._viewport.resize(self.size.width, self.size.height)
3. Add method:
   def on_resize(self, event: Resize) -> None:
       self._viewport.resize(event.size.width, event.size.height)

self._viewport is the ViewportManager. self.size is the Textual widget's current Size (width, height).
The Resize event has event.size with .width and .height.
